Class: NodeDraggable

NodeDraggable(node, snapPx, color, label, showLabel, neuronPxSize, x, y, w, h)

A NodeDraggable contains the information to draw a node in the visual builder. It also has basic functions to move it around by dragging.

Constructor

new NodeDraggable(node, snapPx, color, label, showLabel, neuronPxSize, x, y, w, h)

Create a NodeDraggable
Parameters:
Name Type Description
node Node Reference to the network this Tract belongs to
snapPx number Snap to pixels when dragging
color string Color of the element when drawing
label string Label to display, defaults to last part of path
showLabel boolean Show the label or not
neuronPxSize number How many pixels are used to draw a neuron, determines the w (width) and h (height)
x number The x position of the visual element
y number The y position of the visual element
w number The width of the visual element. Do not change manually, change neuronPxSize
h number The height of the visual element. Do not change manually, change neuronPxSize

move(mouseEvent, parentDOMElementopt)

Move the draggable to a location based as a result of a onmousemove event. Call 'startDragging' first, to save the relative position of the mouse inside the element while dragging.
Parameters:
Name Type Attributes Description
mouseEvent MouseEvent The event returned from the onmousemove trigger

 The HTML element the draggable is drawn in and relative to. Leave empty if this is the parent where the mouse is moving.
